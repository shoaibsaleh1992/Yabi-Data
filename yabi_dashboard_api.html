API Integration Guide

The dashboard fetches data from 5 endpoints. Your backend team needs to implement these routes and return JSON matching the shapes below. Set API_BASE at the top of the script to your server URL.

Base configuration
// Line ~300 in this file — change to your API server const API_BASE = "https://api.yabimoney.com/internal"; const API_KEY = "YOUR_INTERNAL_API_KEY"; // add to request headers
Endpoints
GET/metrics/overview?period=MTD
Top-level KPIs. period is one of MTD | QTD | YTD.

Auth header
// Add this to every request in the fetchData() function headers: { "Authorization": `Bearer ${API_KEY}`, "Content-Type": "application/json" }
While APIs are not yet live
The dashboard runs on realistic sample data. Once your team deploys the endpoints above, change USE_MOCK_DATA = false at the top of the script and the dashboard will switch to live data automatically.
